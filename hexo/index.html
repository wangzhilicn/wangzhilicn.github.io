什么是 Hexo ？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装前提

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

安装 Node.js

Node.js发布于2009年5月，由Ryan Dahl开发，是一个基于Chrome V8引擎的JavaScript运行环境，使用了一个事件驱动、非阻塞式I/O模型， 让JavaScript 运行在服务端的开发平台，它让JavaScript成为与PHP、Python、Perl、Ruby等服务端语言平起平坐的脚本语言。

官网

版本

官网首页目前提供两个版本的下载，分别是 16.16.0 LTS 和 18.5.0 Current 。两个版本的区别在于，前者推荐给大多数用户，后者是最新的版本，看自己需求选择。

安装 Git

Git是一个免费的开源分布式版本控制系统，旨在快速高效地处理从小型到非常大的项目的所有内容。

Git 易于学习，占用空间小，性能快如闪电。它超越了SCM工具，如Subversion，CVS，Perforce和ClearCase，具有廉价的本地分支，方便的暂存区域和多个工作流程等功能。

创建仓库

登录之后点击右上角头像，选择 Your repositories

点击 New 创建一个仓库

  • 如果你没有域名，你可以选择如下设置

注意 Owner 一栏中你的名字，也就是你注册时创建的用户名

Repositories name 填写 用户名.github.io

以我为例就是 hilitech.github.io

这样你托管在 Github 上之后直接在网址栏输入 用户名.github.io 就可以查看你搭建的博客

Repositories name 一栏可以随意填写，在创建完仓库后依次点击 Settings 》 Pages 》Custom domain 》 输入你解析的域名 》 Save 即可

本地生成 Hexo 博客

安装

选择一个文件夹，右键该文件及选择 Git Bash Here

在命令行输入 npm install -g hexo 安装 Hexo

提示更新可以按照提示的内容进行

输入 npm install -g npm@8.13.2 即可更新至最新版本（截至2022/7/10）

查看当前 Hexo 的版本输入 hexo -v

初始化 Hexo

在 Git Bash Here 命令行窗口输入 hexo init 初始化 Hexo 博客

出现 INFO Start blogging with Hexo! 即代表成功

本地启动服务器

在命令行输入 hexo s 启动本地服务器

点击 http://localhost:4000/ 进入，即可查看生成的 Hexo 博客

按 Ctrl + C 结束本地服务器
